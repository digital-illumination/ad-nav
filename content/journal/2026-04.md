## 2026-04-23T14:02Z — Claude Desktop

**Summary:** Built full OAuth 2.1 end-to-end for the remote MCP endpoint over three reviewable chunks: identity foundation (GitHub upstream, Firestore sessions, env-var allowlist), OAuth 2.1 server proper (discovery metadata, dynamic client registration, authorize with PKCE, token endpoint with refresh rotation), and resource server gating (JWT verification and scope enforcement on /api/mcp). Verified the full flow via manual curl before wiring Claude Desktop. Surfaced a likely client-compat issue where write tools return a tool-level error instead of an HTTP 401, which Claude Desktop may not recognise as an OAuth trigger.

**Decisions:**
- Kept the static MCP_WRITE_TOKEN path alongside OAuth rather than replacing it, to avoid churn on the Claude Code config
- Used GitHub as the upstream identity provider with an env-var allowlist, failing closed when the allowlist is empty
- Chose stateless HS256 JWT access tokens and opaque refresh tokens with rotation on use, so validation needs no Firestore round-trip
- Rendered the OAuth consent page as inline HTML from the route handler, matching the site aesthetic without spinning up a React page
- Implemented dynamic client registration so any compliant MCP client can self-register

**Patterns:**
- Ships infrastructure work in small reviewable chunks with a commit per chunk, even when they're small, to keep incremental review easy
- Strong bias for human-in-the-loop on anything touching persistent self-description or auth (allowlist, explicit consent page, human-reviewed promotion from journal to canonical)
- Dogfoods at the wire level with curl before hooking up real clients, so each chunk's contract is proven in isolation

**Follow-ups:**
- Confirm Claude Desktop's OAuth trigger actually fires on first write; if not, return HTTP 401 with WWW-Authenticate on write tool calls when the caller is anonymous
- Rewrite the SPEC.md auth section now that the model is stable (chunk 4 wrap-up)
- Defer the Claude Code Stop-hook autolog until the manual log-session habit has had a few weeks to settle
- Defer the curator agent until the journal accumulates a month or two of material to pattern-match against
